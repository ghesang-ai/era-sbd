📋 Panduan Dashboard SBD Samsung

📤
Upload Data

Klik tombol "Upload Baru" di kanan atas, lalu pilih file Excel SBD Samsung. Data akan langsung diperbarui di semua tab.

📊
Per Tipe

Lihat pencapaian per produk (A37, A57, S26). Setiap kartu menampilkan MTD, estimasi, progress bar, data mingguan, dan top toko.

🎫
Voucher

Monitor utilisasi voucher per produk. Lihat berapa voucher tersedia, terpakai, dan sisa beserta nilai rupiahnya.

👥
Per LOB

Bandingkan performa per Leader (LOB). Lihat grafik MTD dan WoW per leader beserta breakdown A37/A57/S26.

🏪
Per Toko

Tabel detail semua toko. Filter per leader dan sort berdasarkan MTD Total, A37, A57, atau S26.

⭐
Ranking WoW

Top grower dan bottom decline per minggu. Identifikasi toko terbaik dan yang perlu perhatian khusus.

📌 Sumber Data Excel

  • Skema 1 — Target awal per toko
  • Skema 3 — Target per bulan/minggu, alokasi voucher, dan pencapaian aktual
  • Kolom AW–BO di Skema 3 — Recap achievement per TSH/LOB

💡 Tips

  • Upload ulang file setiap minggu untuk update data terbaru
  • Gunakan tombol ⚙ untuk mengubah nama campaign dan banner
  • Klik nama toko di tab Per Toko untuk melihat detail lengkap
